# siray-zapier

Zapier integrations and actions maintained by Siray



## Development 


```bash
# Install dependencies
npm install  # or you can use pnpm or yarn

# Run tests
zapier test

# Register the integration on Zapier if you haven't
zapier register "App Title"

# Or you can link to an existing integration on Zapier
zapier link

# Push it to Zapier
zapier push
```

Then, to add more features, you can use the `zapier scaffold` command, for example:

```bash
# Add a trigger
zapier scaffold trigger contact

# Add an action
zapier scaffold create contact
``` 

## Project Structure

```bash
siray-zapier/
│
├── package.json                  # Node.js deps: zapier-platform-core
├── .zapierapprc                  # App ID + deployment  config (generated by CLI)  => moved to user `C:\Users\{user}`
├── index.js                      # ROOT: registers all triggers/actions/searches
│
├── authentication.js             # API Key + Base URL auth config
│
├── triggers/
│   ├── videoGenerationCompleted.js   # REST Hook — video async task done
│   └── imageGenerationCompleted.js   # REST Hook — image async task done
│
├── action/                      # Zapier calls these "actions" in the UI
│   ├── chatCompletion.js             # Sync LLM inference
│   ├── generateVideo.js              # Submit async video task
│   ├── generateImage.js              # Submit async image task
│   └── getTaskStatus.js              # Poll task by ID
│
├── searches/
│   └── modelList.js                  # Dynamic dropdown for model selection
│
└── test/
    ├── authentication.test.js
    ├── triggers/
    │   ├── videoGenerationCompleted.test.js
    │   └── imageGenerationCompleted.test.js
    ├── creates/
    │   ├── chatCompletion.test.js
    │   ├── generateVideo.test.js
    │   ├── generateImage.test.js
    │   └── getTaskStatus.test.js
    └── searches/
        └── modelList.test.js


    ```
